# ☕ Coffee OS (ZoneBarista Knowledge Base)

> **An ultra-aesthetic, serverless, and cinematic knowledge explorer designed specifically for the Coffee Intelligence Database.**

![Coffee OS Banner](https://images.unsplash.com/photo-1497935586351-b67a49e012bf?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

Coffee OS is a premium, beautifully crafted frontend interface built to dynamically parse, render, and navigate markdown-based knowledge bases. Built with a hand-drawn, "warm paper sketch" aesthetic, it transforms raw data into a luxurious, Apple-esque scientific archive.

## ✨ Features

- **🚀 100% Serverless Architecture**: Runs entirely in the browser using raw HTML, CSS, and JS. Zero backend required.
- **🎨 Cinematic UI/UX**: Warm paper textures, wobbly borders, glassmorphism overlays, and micro-animations.
- **📈 Advanced Rendering**: Built-in, automatic support for Markdown, `KaTeX` (Math), `Highlight.js` (Code), and `Mermaid.js v10` (Advanced Flowcharts & Mindmaps).
- **⚡ Lightning Fast Search**: Integrated `Fuse.js` for instant, fuzzy-search capabilities across the entire knowledge base.
- **📱 Fully Responsive**: Flawless experience on desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- **Core**: Vanilla HTML5, CSS3, ES6 JavaScript
- **Markdown Parsing**: Marked.js
- **Icons**: Lucide & Custom inline SVGs
- **Diagrams**: Mermaid.js (Asynchronous rendering to bypass DOM entity bugs)
- **Math/Formulas**: KaTeX
- **Search**: Fuse.js

## 🚀 Getting Started

Since Coffee OS is completely serverless, there is no build step or package manager required to run it!

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start exploring!

*(Note: For local testing with `file:///`, some browsers restrict fetching local files due to CORS. If you see an error, use a local server like `npx serve` or VS Code Live Server, OR just deploy it live!)*
