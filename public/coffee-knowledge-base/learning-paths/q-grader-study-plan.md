## What Is a Q Grader?

The **Q Arabica Grader** is the highest individual professional sensory certification in the coffee industry, issued by the **Coffee Quality Institute (CQI)**.

A certified Q Grader can:
- Evaluate Arabica coffee to SCA standards
- Score to SCA cupping protocol reliably and calibrate with other Q Graders
- Grade green coffee for defects
- Identify individual organic acids in solution
- Detect roast colour to Agtron standards
- Pass 22 individual module tests
- Recertify every 3 years (calibration requirement)

**Passing rate:** Approximately 50% on first attempt — rigorous examination.

## 6-Month Study Programme

### Month 1: Foundation & Knowledge

**Week 1–2: Coffee Fundamentals**
```
□ Read: history-of-coffee.md
□ Read: specialty-coffee-movement.md
□ Read: supply-chain.md
□ Read: certifications-standards.md
□ Read: beans/species-overview.md
□ Study: All origin regions (Ethiopia, Brazil, Colombia, India, Kenya, Guatemala, Indonesia)
□ Flash cards: origin → flavour profile → processing → altitude
□ Practice: Name all major varietals and their cup characteristics
```

**Week 3–4: Roasting & Processing**
```
□ Read: roasting/roast-science.md
□ Read: roasting/roast-curves-profiles.md
□ Read: roasting/roast-defects.md
□ Read: beans/green-coffee-grading.md

□ Study: All primary and secondary defects (names + visual ID)
□ Purchase Agtron sample tiles or find practice set
```

### Month 2: Cupping Mastery

**Daily cupping practice (30 min minimum):**
```
□ Cup 2–3 coffees per day to SCA protocol
□ Write full tasting notes for each (all 10 attributes)
□ Compare your scores with lead Q Grader / training partner
□ Track score variance from "expected" (build internal reference)
□ Read: cupping-protocol.md (master every detail)
□ Read: flavor-wheel-guide.md (memorise all 9 primary categories)
```

**Weekly calibration sessions:**
```
□ Cup with certified Q Grader or instructor weekly
□ Focus: staying within 2 points on all cups
□ Identify your systematic biases (do you consistently score acidity low?)
□ Adjust mental calibration accordingly
```

**Cupping calibration tracking:**
```
SESSION LOG:
Date: ________  Coffee A: Q Grader: ____  Mine: ____  Diff: ____
               Coffee B: Q Grader: ____  Mine: ____  Diff: ____
               Coffee C: Q Grader: ____  Mine: ____  Diff: ____
Avg variance: ____  Target: < 1.0 pts average
```

### Month 3: Sensory Threshold Training

**Basic Taste Threshold Solutions:**

Prepare these in distilled water for threshold training:

| Compound | Threshold Solution | Moderate | Strong |
|---------|-------------------|---------|--------|
| Sucrose | 1g/L | 5g/L | 20g/L |
| Citric acid | 0.1g/L | 0.5g/L | 2g/L |
| Caffeine | 0.1g/L | 0.4g/L | 1.5g/L |
| NaCl (salt) | 0.3g/L | 1g/L | 3g/L |

```
DAILY THRESHOLD EXERCISE (15 min):
1. Prepare 4 solutions at different concentrations each compound
2. Taste blind; rank from lowest to highest intensity
3. Identify compound (sweet/sour/bitter/salty)
4. Note your detection threshold (lowest you can reliably detect)
5. Track improvement over weeks
```

**Organic Acid Training:**

Prepare 1g/L solutions of each:
- Citric acid → Lemon, citrus
- Malic acid → Apple, clean
- Acetic acid → Vinegar, fermented
- Phosphoric acid → Crisp, mineral, clean
- Tartaric acid → Grape, wine
- Lactic acid → Yogurt, creamy, smooth

```
ACID TRAINING PROTOCOL (10 min daily):
1. Label 6 beakers A–F (unknown acids)
2. Taste each; describe in own words
3. Match to known compound
4. Check answers
5. Note any still confused (focus next session)
```

### Month 4: Aroma Training

**Le Nez du Café — 36 Aroma Reference Kit**

The kit contains physical reference standards for all 36 primary coffee aromas, organised in 3 groups:

**Enzymatic Group (green/light roast aromas):**
```
Aromatics: Blackcurrant-like, Garden peas, Cucumber, Unripe,
           Garden-like (geranium-like), Jasmine, Tea rose,
           Coffee blossom
```

**Sugar-Browning Group (Maillard/medium roast):**
```
Aromatics: Roasted almonds, Walnuts, Roasted peanuts,
           Dark chocolate, Caramel, Butter, Toast,
           Fresh bread, Honey, Coffee essence, Malt
```

**Dry Distillation Group (dark roast/pyrolysis):**
```
Aromatics: Pepper, Coriander seeds, Clove, Spicy, Pipe tobacco,
           Cedar, Earthy, Leather, Medicinal, Rubber, Skunky,
           Chemical
```

**Daily Aroma Protocol:**
```
SESSION 1 (Days 1–30): Identification by group
  - Learn one group per 10 days
  - Smell each vial; read label; memorise association
  - Repeat 3× per vial per session

SESSION 2 (Days 31–60): Blind identification
  - Smell 5 vials blind; name each
  - 80% correct target before moving on

SESSION 3 (Days 61–90): Triangle testing with aromas
  - 3 vials: 2 same, 1 different
  - Must identify the odd one out
  - Target: 7/8 correct consistently
```

### Month 5: Triangle Testing & Green Grading

**Triangle Test Practice (daily, 20 min):**

```
PROGRESSIVE DIFFICULTY LADDER:

EASY (start here):
  Ethiopian natural vs Colombian washed vs Brazilian natural
  → Very distinct origins; processing-driven differences

MEDIUM (weeks 2–3):
  Colombian Huila vs Colombian Nariño (same country, different region)
  → Requires developed palate

HARD (weeks 3–4):
  Same coffee, different extraction levels (under vs optimal vs over)
  → Pure technique discrimination

EXPERT (week 5+):
  Same coffee, different roast dates (fresh vs 2 weeks vs 5 weeks)
  → Freshness discrimination
```

**Green Coffee Grading Practice:**

```
EQUIPMENT NEEDED:
□ Precision scale (0.1g)
□ Grading tray (white, flat)
□ Sample bags of defective green coffee (purchase from CQI or specialty importers)

PROCEDURE (practice each session):
1. Weigh 350g of green coffee
2. Spread on grading tray
3. Sort through bean by bean
4. Separate all defects by category
5. Apply equivalence table
6. Total: primary defect count + secondary defect count
7. Determine grade
8. Compare to answer key (if using practice samples)

TARGET ACCURACY: Within ±1 defect count of standard answer
```

### Month 6: Mock Exams & Final Calibration

**Full mock exam weeks:**

```
WEEK 1: Knowledge modules (written)
  - Full written exam under timed conditions
  - Review all incorrect answers
  - Re-read relevant files

WEEK 2: Sensory threshold mock
  - Full threshold testing session
  - Full acid identification session
  - Full aroma kit (all 36 blind)

WEEK 3: Cupping calibration
  - 5 cupping sessions this week
  - Cup with Q Grader for every session
  - Target: within 2 pts on all cups in all sessions

WEEK 4: Triangle + green grading
  - 4 triangle sets per day
  - 1 green grading session per day
  - Roast ID: match 5 samples to Agtron tiles blind
```

## Recertification

Q Grader certification must be renewed every **3 years**:

| Renewal Option | Description |
|---------------|-------------|
| **Full re-examination** | Retake all 22 modules |
| **Calibration session** | Abbreviated cupping calibration with CQI-approved Q Instructor |
| **Online calibration** | New option: remote calibration modules (check current CQI policy) |

**Important:** If calibration lapses, certification becomes inactive. Active Q Graders who cup regularly rarely fail recertification.
