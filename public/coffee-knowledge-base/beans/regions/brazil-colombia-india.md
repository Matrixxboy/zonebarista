# Brazil — Coffee Origin Profile

## Country Overview

| Parameter | Data |
|-----------|------|
| Production Rank | **#1 globally** |
| Annual Production | ~55–65 million 60kg bags |
| Primary Species | Arabica (~75%), Robusta/Conillon (~25%) |
| Altitude Range | 600–1,300 masl (low compared to other origins) |
| Harvest Season | May–September |
| Processing | Natural, Pulped Natural (Honey), Washed |
| Dominant Use | Blends, espresso, mass market + specialty |

> 🌍 Brazil produces roughly **1/3 of all coffee on Earth** and is the largest exporter to both commodity and specialty markets.

---

## Key Regions

| Region | State | Altitude (masl) | Flavor Profile |
|--------|-------|----------------|----------------|
| **Sul de Minas** | Minas Gerais | 800–1,300 | Chocolate, nuts, mild citrus, clean |
| **Cerrado Mineiro** | Minas Gerais | 800–1,200 | Caramel, dark chocolate, low acid, full body |
| **Chapada Diamantina** | Bahia | 900–1,200 | Tropical fruit, caramel, sweet |
| **Mogiana** | São Paulo | 900–1,200 | Balanced, nutty, caramel |
| **Espírito Santo** | ES | 600–800 | Robusta/Conillon grown here |
| **Alta Mogiana** | São Paulo | 1,000–1,300 | Higher quality, fruity, balanced |

---

## Key Varietals in Brazil

| Varietal | Characteristics |
|---------|----------------|
| **Bourbon** | Classic cup quality; fruity, sweet; lower yield |
| **Catuai** | High productivity; balanced, nutty, mild |
| **Mundo Novo** | Full body, chocolate; excellent blending |
| **Acaiá** | Excellent cup, lower yield |
| **Yellow Bourbon** | Sweetness, tropical fruit notes |
| **Icatu** | Robusta × Bourbon hybrid; disease resistant |
| **Catucaí** | Hybrid for harsh climates |

---

## Processing in Brazil

Brazil pioneered **pulped natural / honey processing**:

| Method | Description | Cup Impact |
|--------|-------------|-----------|
| **Natural** | Whole cherry dried | Fruity, wine-like, heavy body |
| **Pulped Natural** | Pulp removed; mucilage retained | Balanced; chocolate, caramel, medium body |
| **Washed** | Less common | Cleaner but less characteristic |

---

## Flavor Profile Summary

```
Brazilian Specialty (typical):
  Nutty/Cocoa → Hazelnut → ✅
  Nutty/Cocoa → Dark Chocolate → ✅
  Sweet → Caramel → ✅
  Sweet → Brown Sugar → ✅
  Fruity → Berry → (natural processing) → ✅
  Low to medium acidity
  Heavy, syrupy body
```

---

## Roast Compatibility

Brazil is extremely **roast-versatile**:
- **Light roast:** Reveals caramel, fruit, nougat
- **Medium roast:** Classic balance — nut, chocolate, caramel
- **Dark roast:** Bittersweet chocolate, good crema base
- **Espresso:** One of the best single-origin espresso origins (or blend base)

# Colombia — Coffee Origin Profile

## Country Overview

| Parameter | Data |
|-----------|------|
| Production Rank | #3 globally |
| Annual Production | ~12–14 million 60kg bags |
| Primary Species | 100% Arabica |
| Altitude Range | 1,200–2,100 masl |
| Harvest Season | Main: Oct–Feb; Mitaca (second): Apr–Jun |

## Key Regions

| Region | Altitude (masl) | Flavor Profile |
|--------|----------------|----------------|
| **Huila** | 1,500–2,100 | Red fruit, citrus, floral, bright acidity |
| **Nariño** | 1,700–2,300 | Citrus, stone fruit, tea-like, high acidity |
| **Cauca** | 1,700–2,100 | Clean, sweet, fruity, balanced |
| **Antioquia** | 1,200–2,000 | Balanced, chocolate, mild fruit |
| **Sierra Nevada** | 1,000–2,000 | Balanced, chocolate, low acidity |
| **Santander** | 1,000–1,800 | Nutty, chocolate, mild |

---

## Key Varietals

| Varietal | Notes |
|---------|-------|
| **Caturra** | Colombia's primary varietal; bright, clean |
| **Castillo** | FNC hybrid; resistant to CLR; clean cup |
| **Colombia** | Hybrid; disease resistant |
| **Bourbon** | Traditional; sweet, complex; lower yield |
| **Tabi** | New FNC variety; excellent cup, tall plant |

---

## Cup Profile

```
Colombian Specialty (typical):
  Fruity → Red Fruit → Red Apple, Cherry → ✅
  Fruity → Citrus → Orange → ✅
  Sweet → Caramel → ✅
  Nutty/Cocoa → Milk Chocolate → ✅
  Medium acidity (malic/citric)
  Medium body
  Very clean finish
```

# India — Coffee Origin Profile

---
title: India — Coffee Origin Profile
tags: [india, coorg, chikmagalur, araku, monsooned-malabar, robusta, arabica]
---

## Country Overview

| Parameter | Data |
|-----------|------|
| Production Rank | 6th–7th globally |
| Annual Production | ~5–6 million 60kg bags |
| Primary Species | Robusta (~70%), Arabica (~30%) |
| Altitude Range | 600–1,800 masl |
| Harvest Season | November–February |
| Processing | Washed, Natural, Wet-hulled (Monsooned) |
| Regulatory Body | Coffee Board of India |

---

## Key Regions

| Region | State | Species | Flavor Profile |
|--------|-------|---------|----------------|
| **Coorg (Kodagu)** | Karnataka | Arabica + Robusta | Spice, chocolate, earthy |
| **Chikmagalur** | Karnataka | Arabica | Fruity, medium acidity, clean |
| **Wayanad** | Kerala | Robusta dominant | Earthy, woody, chocolate |
| **Nilgiris** | Tamil Nadu | Arabica | Floral, mild, bright |
| **Araku Valley** | Andhra Pradesh | Arabica | Chocolate, mild spice, clean (tribal-grown) |
| **Bababudangiri** | Karnataka | Arabica | Balanced, historical origin area |

> ☕ **Bababudangiri** (also Baba Budan Hills) is where Baba Budan allegedly smuggled 7 coffee beans from Yemen to India in the 17th century — the origin of Indian coffee cultivation.

---

## Monsooned Malabar — Unique Indian Process

| Attribute | Detail |
|-----------|--------|
| Origin | Malabar Coast (Karnataka/Kerala) |
| Process | Green beans exposed to monsoon winds for 12–16 weeks |
| Bean Change | Swells to 2× normal size; loses acidity; color turns pale gold/yellow |
| Cup Profile | Very heavy body, almost no acidity, earthy, woody, musty, tobacco, spice |
| Common Use | Espresso blends (adds body/crema); traditional South Indian filter coffee |

---

## South Indian Filter Coffee

A cultural institution:
- **Blend:** 80% Robusta, 20% Arabica (or Chicory)
- **Brewing:** Traditional metal filter (percolation), served with hot milk in tumbler-davara set
- **Signature:** Sweet, milky, strong, frothy — poured between vessels to aerate

---

## Indian Coffee in Specialty

India is growing rapidly in specialty:
- **Araku Valley** coffees (tribal-grown, organic) winning international recognition
- **Yercaud** (Tamil Nadu) producing clean Arabica specialty
- Emerging **anaerobic and experimental processing** in Karnataka
- **Indian Coffee Specialists** (ICS) Q Grader programs expanding
