## Taxonomy Note

Excelsa was reclassified in **2006** (Davis et al., Kew Bulletin) from its own species (*Coffea dewevrei* De Wild. & T.Durand) to a **variety of Liberica**:

> **Current name:** *Coffea liberica* W.Bull ex Hiern var. *dewevrei* (De Wild. & T.Durand) Chevalier

It is still commonly called "Excelsa" in trade and among farmers.

---

## Physical Characteristics

| Feature | Description |
|---------|-------------|
| Bean shape | Elongated, almond-like; more symmetrical than Liberica |
| Bean size | Medium-large |
| Cherry | Smaller than Liberica |
| Tree height | 7–12m tall |
| Leaf | Large, glossy |
| Chromosome number | 2n = 22 (diploid) |

## Agronomic Profile

| Parameter | Value |
|---------|-------|
| Altitude | 0–800 masl |
| Origin | Central Africa (Congo Basin) |
| Primary cultivation | Philippines, Vietnam, some Central Africa |
| Disease resistance | High (similar to Liberica) |
| Drought tolerance | Good |
| Yield | Moderate |

---

## Sensory Profile

| Attribute | Description |
|---------|-------------|
| Aroma | Dark fruit, tart, wine-like, complex |
| Flavor | **Complex contrast:** tart + dark fruit + roasty depth; sometimes smoky |
| Body | Medium |
| Acidity | Medium–high (more than Liberica; less than bright Arabica) |
| Sweetness | Low |
| Finish | Long, complex, slightly astringent |
| Key character | Simultaneous tartness and depth — unlike any other species |

---

## Commercial Use

| Use | Detail |
|-----|--------|
| **Philippine blends** | Added to "Filipino blend" coffees for complexity and tartness |
| **Vietnamese blends** | Small percentage alongside Robusta in some brands |
| **Specialty exploration** | Growing curiosity from adventurous specialty roasters |

Excelsa is valued in blends for:
- Adding **complexity** that neither Arabica nor Robusta provides
- Contributing **dark fruit and tart** dimension
- Its **distinctive character** at low percentages (10–20%)

---

## Comparison: Liberica vs Excelsa

| Attribute | Liberica | Excelsa |
|---------|---------|---------|
| Bean size | Very large, irregular | Medium-large, more uniform |
| Flavor | Smoky, woody, floral | Tart, dark fruit, wine |
| Body | Very heavy | Medium |
| Acidity | Low | Medium–high |
| Aroma | Unusual, floral-smoky | Dark fruit, complex |
| Use | Local consumption, niche | Blending, niche |
