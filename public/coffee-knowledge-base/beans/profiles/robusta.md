## Scientific Classification

| Taxonomy | Value |
|---------|-------|
| Kingdom | Plantae |
| Family | Rubiaceae |
| Genus | *Coffea* |
| Species | *Coffea canephora* Pierre ex A.Froehner |
| Common variety | *C. canephora* var. *robusta* (name now used generically) |
| Chromosome number | 2n = 22 (diploid) |
| Origin | Western/Central Africa (Congo River Basin, Uganda) |
| Pollination | Cross-pollination required (self-sterile) |

---

## Physical Characteristics

| Feature | Description |
|---------|-------------|
| Bean shape | Round, more symmetric than Arabica |
| Bean size | Smaller than Arabica; screen 13–17 |
| Center cut | Straight (vs S-curved in Arabica) |
| Cherry | Smaller; tighter clusters |
| Tree height | 2–10m; more robust growth habit |
| Root system | Shallower than Arabica |
| Green color | Pale, less blue-green than Arabica |

---

## Agronomic Profile

| Parameter | Value |
|---------|-------|
| Altitude | 0–800 masl (some high-altitude: 1,200+ masl in Uganda/India) |
| Temperature | 22–30°C optimal (more heat-tolerant than Arabica) |
| Rainfall | 2,000–2,500mm/year |
| Disease resistance | **High** — natural pest/disease resistance (higher caffeine = insect deterrent) |
| Drought tolerance | Moderate |
| Yield | 2–4× higher than Arabica per tree |
| Time to first harvest | 2–3 years |
| Productive lifespan | 20–30 years |

---

## Chemical Composition vs Arabica

| Compound | Robusta (%) | Arabica (%) | Impact |
|---------|------------|-------------|--------|
| Caffeine | 2.2–2.7 | 1.2–1.5 | Bitterness; stimulant; pest deterrence |
| Chlorogenic acids | 7–10 | 5.5–8 | Astringency; antioxidants |
| Sucrose | 3–7 | 6–9 | Less sweetness potential |
| Lipids | 10–11.5 | 15–17 | Less body; better crema emulsification |
| Trigonelline | 0.3–0.7 | 0.6–1.0 | Less aroma precursor |

> 🔬 *The higher CGA and caffeine content in Robusta directly contributes to its harsher, more bitter profile. The lower sucrose means less Maillard sweetness development during roasting.*

## Robusta's Role in Espresso Blending

Italian espresso tradition uses **10–30% Robusta** for:

| Benefit | Mechanism |
|---------|-----------|
| **Crema** | Higher melanoidin + emulsifying agent content → thicker, more persistent crema |
| **Body** | Lower lipid but higher suspended solids → heavy mouthfeel |
| **Caffeine** | Bold kick per shot |
| **Cost** | Lowers blend price significantly |
| **Boldness** | Strong base note cuts through milk |

**Classic Italian espresso blend:** 70–80% Arabica (Brazil, Colombia) + 20–30% Robusta (Vietnam, India, Uganda)

## Fine Robusta / Specialty Robusta

An emerging category where high-altitude, well-processed Robusta scores 80+ on adapted cupping scales:

| Region | Key Attributes |
|--------|---------------|
| **Uganda (Rwenzori, Sipi Falls)** | 1,200–1,800 masl; clean, mild, chocolate |
| **India (Wayanad, Coorg)** | Shade-grown; spice, chocolate, earthy |
| **Vietnam (Da Lat highlands)** | Emerging specialty; mild, balanced |
| **Ivory Coast** | Traditional quality Robusta |

**CABI/SCA Fine Robusta Standards:** Adapted cupping protocols for Robusta are under development. Fine Robusta typically scores 80+ on a modified 100-point scale that accounts for species differences.

## Vietnamese Coffee Culture

Vietnam produces ~95% Robusta and has a distinct national coffee culture:

| Drink | Description |
|-------|-------------|
| **Cà phê sữa đá** | Iced Robusta drip + sweetened condensed milk |
| **Cà phê trứng** | Espresso-style Robusta + whipped egg yolk foam |
| **Cà phê muối** | Robusta + salted cream (Huế style) |
| **Cà phê dừa** | Robusta + coconut milk |

**Brewing tool:** **Phin filter** — a traditional Vietnamese metal drip filter that brews slow-drip, concentrated Robusta coffee.
