## *Coffea liberica* — Profile

### Scientific Classification

| Taxonomy | Value |
|---------|-------|
| Species | *Coffea liberica* W.Bull ex Hiern |
| Common name | Liberica |
| Chromosome number | 2n = 22 (diploid) |
| Origin | Western Africa (Liberia) |
| First cultivation outside Africa | Philippines (1876, by Spanish colonists) |

### Physical Characteristics

| Feature | Description |
|---------|-------------|
| Bean size | **Largest of all species** — ~2× Arabica size |
| Bean shape | Irregular, often asymmetric, hook-like |
| Cherry size | Very large |
| Tree height | 9–18m (very tall in wild) |
| Leaf size | Large, waxy |

### Why Liberica Matters Historically

In the 1880s, **Coffee Leaf Rust** (*Hemileia vastatrix*) devastated Arabica plantations across Southeast Asia. The British and Dutch introduced Liberica as a replacement:
- **Malaysia:** Became primary species for a period
- **Philippines:** Liberica (Barako) took root in Batangas region — still culturally significant

### Cup Profile

| Attribute | Description |
|---------|-------------|
| Aroma | Floral, fruity, then woody, smoky — **very distinctive** |
| Flavor | Unusual — smoky, woody, floral, sometimes jackfruit-like, musty |
| Body | Full, heavy |
| Acidity | Low |
| Finish | Long, lingering, earthy |
| Overall character | Polarizing — unlike Arabica or Robusta |

### Kapeng Barako (Philippines)

The most famous Liberica expression:
- Grown in **Batangas and Cavite** provinces, Luzon
- Traditional Philippine coffee culture centerpiece
- Served very strong with sugar; bold, full-bodied
- Critically endangered variety locally due to economic pressure

### Commercial Significance

- **< 1% of global production**
- Limited to specific regions: Philippines, Malaysia, Indonesia, parts of West Africa
- Not traded internationally at scale
- Growing niche interest in specialty market

## *Coffea liberica* var. *dewevrei* — Excelsa Profile

### Taxonomy Note

Excelsa was reclassified in 2006 (Davis et al.) from its own species (*C. dewevrei*) to a **variety of Liberica** (*C. liberica* var. *dewevrei*). It's still commonly called "Excelsa" in trade.

### Physical Characteristics

| Feature | Description |
|---------|-------------|
| Bean shape | Elongated, almond-like |
| Bean size | Medium-large |
| Tree height | Tall (similar to Liberica) |
| Cultivation | Philippines, Vietnam, Central Africa |

### Cup Profile

| Attribute | Description |
|---------|-------------|
| Aroma | Dark fruit, tart, wine-like |
| Flavor | **Complex:** tart, fruity, dark, sometimes smoky; wine and fruit notes |
| Body | Medium |
| Acidity | Medium–high (more than Liberica) |
| Aftertaste | Long, complex |
| Use | **Primarily in blends** — adds complexity and dark fruit dimension |

### Commercial Use

- Philippines: Used in blended "Filipino blend" coffees
- Vietnam: Small percentage in some blends
- Adding depth and complexity to commodity blends
- Growing attention from specialty industry for experimentation
