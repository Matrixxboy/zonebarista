## Scientific Classification

| Taxonomy | Value |
|---------|-------|
| Kingdom | Plantae |
| Family | Rubiaceae |
| Genus | *Coffea* |
| Species | *Coffea arabica* L. |
| Chromosome number | 2n = 44 (allotetraploid) |
| Origin | Ethiopian Highlands / South Sudan |
| Natural hybridization parents | *C. canephora* × *C. eugenioides* |
| Discovered by science | 1753 (Linnaeus) |

> 🔬 *Arabica is unique: it's the result of a natural polyploidization event between two diploid species (C. canephora, 2n=22, and C. eugenioides, 2n=22), resulting in a tetraploid with 2n=44. This hybrid origin gives Arabica its genetic complexity and flavor richness.*

## Agronomic Profile

| Parameter | Value |
|---------|-------|
| Altitude preference | 600–2,200 masl |
| Temperature | 15–24°C optimal |
| Rainfall | 1,500–2,500mm/year |
| Soil | Well-drained, volcanic preferred; pH 6.0–6.5 |
| Self-fertile | Yes (pollination doesn't require cross) |
| Tree height | 2–5m (cultivated, pruned to 2–3m) |
| Time to first harvest | 3–4 years from seedling |
| Productive lifespan | 20–30 years |
| Yield | 0.5–5kg cherry/tree/year (variety-dependent) |
| Major disease | Coffee Leaf Rust (*Hemileia vastatrix*) — highly susceptible |

## Chemical Composition (Green Bean)

| Compound | % Dry Weight | Role in Cup |
|---------|-------------|-------------|
| Sucrose | 6–9% | Sweetness, caramelization during roast |
| Reducing sugars | 0.1–0.5% | Maillard reaction, aroma |
| Lipids (fats) | 15–17% | Body, mouthfeel, crema |
| Chlorogenic acids (CGAs) | 5.5–8% | Astringency, acidity; degrade during roasting |
| Caffeine | 1.2–1.5% | Bitterness, stimulant |
| Trigonelline | 0.6–1.0% | Aroma precursor; bitter in isolation |
| Proteins | 11–13% | Maillard reaction, body |
| Polysaccharides | 50%+ | Structure, body |

# Robusta — Complete Species Profile

## Scientific Classification

| Taxonomy | Value |
|---------|-------|
| Species | *Coffea canephora* Pierre ex A.Froehner |
| Common name | Robusta (most common variety of canephora) |
| Chromosome number | 2n = 22 (diploid) |
| Origin | Western/Central Africa (Congo River Basin) |
| Self-fertile | No — requires cross-pollination |

---

## Key Differences vs Arabica

| Attribute | Robusta | Arabica |
|---------|---------|---------|
| Chromosomes | 22 | 44 |
| Caffeine | 2.2–2.7% | 1.2–1.5% |
| Chlorogenic acids | 7–10% | 5.5–8% |
| Lipids | 10–11.5% | 15–17% |
| Sucrose | 3–7% | 6–9% |
| Disease resistance | High | Low |
| Altitude preference | 0–800 masl | 600–2200 masl |
| Flavor | Harsh, earthy | Complex, fruity/floral |
| Crema production | More (more emulsifying agents) | Less (comparatively) |
| Market price | Lower | Higher |

---

## Why Robusta in Espresso Blends?

Italian espresso tradition uses 10–30% Robusta for:
1. **Crema:** Robusta produces thicker, more persistent crema (higher lipid + melanoidin content)
2. **Body:** Heavier mouthfeel
3. **Cost:** Lower price point in blends
4. **Caffeine:** Higher caffeine appeal in some markets
5. **Boldness:** Earthy, intense base note

> ✅ *High-quality ("Fine Robusta") from Uganda, India, or Vietnam highlands can cup at 80+ on adapted scales. Not all Robusta is bad Robusta.*

## Vietnamese Coffee Culture

Vietnam (world's #2 producer, ~95% Robusta) has its own distinct coffee culture:
- **Cà phê trứng** (egg coffee): Robusta espresso with whipped egg yolk
- **Cà phê sữa đá**: Iced Robusta with sweetened condensed milk
- **Phin filter**: Traditional metal drip filter brewer
- Flavor suits the sweet/dairy additions due to bold bitterness
