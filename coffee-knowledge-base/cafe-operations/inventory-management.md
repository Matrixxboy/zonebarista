## Why Inventory Management Matters

Poor inventory control is one of the top causes of café failure:
- **Over-ordering** → stale coffee, spoiled milk, cash tied up in stock
- **Under-ordering** → running out mid-service, unhappy customers, emergency orders at premium prices
- **No tracking** → unknown waste, undetected theft, inaccurate COGS
- **Poor rotation** → serving stale product, quality complaints

---

## The FIFO Principle

**FIFO = First In, First Out**

Always use the oldest stock first:

```
Shelf arrangement:
NEW STOCK → [Back of shelf] ─────────────────────→ [Front of shelf] → USE FIRST

Rotation protocol:
1. New delivery arrives
2. Move existing stock to FRONT
3. Place new stock at BACK
4. Staff always take from FRONT
```

Apply to: coffee beans, milk, syrups, food items, packaging.

## Par Stock System

**Par level** = the minimum quantity that must be on hand at all times, accounting for lead time and safety buffer.

### Par Stock Formula

$$\text{Par Level} = (\text{Daily Usage} \times \text{Lead Time Days}) + \text{Safety Stock}$$

$$\text{Safety Stock} = \text{Daily Usage} \times \text{Buffer Days (1–3)}$$

### Coffee Bean Par Stock Example

| Input | Value |
|-------|-------|
| Daily shots pulled | 200 |
| Dose per shot | 18g |
| Daily coffee usage | 200 × 18g = 3,600g = 3.6kg |
| Supplier lead time | 3 days |
| Safety buffer | 2 days |
| **Par level** | 3.6 × (3+2) = **18kg minimum on hand** |

**Reorder point:** Place order when stock falls to par level.

---

### Milk Par Stock Example

| Input | Value |
|-------|-------|
| Milk drinks per day | 150 |
| Average milk per drink | 200mL |
| Daily milk usage | 150 × 200mL = 30L |
| Delivery frequency | 3× per week (Mon/Wed/Fri) |
| Max days between delivery | 2 days |
| Safety buffer | 1 day |
| **Par level** | 30L × (2+1) = **90L minimum** |

## Ordering Schedule Template

### Weekly Ordering Calendar

| Day | Action | Items |
|-----|--------|-------|
| **Monday** | Count stock; place weekly order | Coffee beans, packaging, syrups |
| **Tuesday** | Receive dry goods order | Inspect, count, sign invoice |
| **Wednesday** | Mid-week milk top-up | Dairy, plant milks |
| **Thursday** | Review week's stock levels | Flag any shortfalls |
| **Friday** | Pre-weekend stock check | Ensure weekend par covered |
| **Saturday** | Spot check high-use items | Milk, cups, coffee |
| **Sunday** | Weekly waste log completion | Record all waste |

## Waste Tracking System

### Daily Waste Log

```
DATE: ___________

COFFEE WASTE
  Dial-in shots (discarded): _____ shots × 18g = _____ g
  Quality failures/remakes:  _____ shots × 18g = _____ g
  Batch brew discarded:      _____ L × ___g/L  = _____ g
  Grinder purge:             _____ g
  TOTAL COFFEE WASTE:        _____ g
  COST OF WASTE:             _____ g × $___/g  = $_____

MILK WASTE
  Over-steamed/discarded:    _____ mL
  Expired/disposed:          _____ mL
  Remake waste:              _____ mL
  TOTAL MILK WASTE:          _____ mL
  COST:                      $_____

FOOD/OTHER
  [Item]:                    _____ units discarded
  Cost:                      $_____

TOTAL DAILY WASTE COST:      $_____
TARGET (< 3% of daily COGS): $_____
```

### Waste % Calculation

$$\text{Waste \%} = \frac{\text{Waste Cost}}{\text{Total COGS}} \times 100$$

**Target:** Coffee waste < 3% of daily coffee COGS  
**Alert level:** > 5% — investigate immediately

## Waste Reduction Strategies

### Coffee Waste

| Source | Strategy | Target Saving |
|--------|---------|--------------|
| Dial-in waste | Dial in before service; use notes from previous day | 30–50% reduction |
| Batch brew over-production | Brew smaller batches more frequently | 20–40% reduction |
| Grinder retention | Use single-dose grinder or purge less frequently | 10–20% reduction |
| Quality remakes | Better training; consistent workflow | 15–30% reduction |
| Expired stock | FIFO + smaller orders more frequently | Near elimination |

### Milk Waste

| Source | Strategy |
|--------|---------|
| Over-steaming | Match pitcher size to drink; steam to order |
| Re-steaming (not allowed) | Train staff: discard, don't reheat |
| Expired milk | Order frequency matched to turnover; FIFO always |
| Wrong drink remakes | Order confirmation protocol |

## Supplier Management

### Key Supplier Categories

| Category | Frequency | Key Metrics |
|---------|-----------|------------|
| **Coffee (roaster)** | Weekly–bi-weekly | Freshness window; roast date; lead time |
| **Dairy** | 3× per week | Shelf life; cold chain; volume |
| **Plant milks** | Weekly | Ambient storage; shelf life; volume |
| **Syrups/flavourings** | Monthly | Shelf life; volume; storage |
| **Packaging** | Monthly | Lead time; storage space; minimum order |
| **Equipment consumables** | Quarterly | Filter cartridges; cleaning products |

### Supplier Performance Scorecard

| Metric | Measure | Target |
|--------|---------|--------|
| On-time delivery | % orders delivered on agreed day | > 95% |
| Order accuracy | % orders correct | > 98% |
| Quality consistency | Sensory score variance | < 2 points |
| Responsiveness | Time to resolve issues | < 24 hours |
| Communication | Proactive notification of shortfalls | Yes |
